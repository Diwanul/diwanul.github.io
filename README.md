# diwanul.github.io

This repository contains the source code for my personal academic homepage.

## Structure Overview

*   `index.html`: The main content file. You can edit this file to update your bio, education, publication list, and links.
*   `css/style.css`: Contains minimal, clean styling to ensure readability and maintain an academic look.
*   `images/` (To be created): You can create this folder to store your profile photo (`profile.jpg`) or any other assets.

## How to edit

1.  Open `index.html`.
2.  Replace the placeholder text (e.g., `[Your University/Institute]`, `[Paper Title]`) with your actual information.
3.  Add your own photo inside an `images` directory and update the `img` tag placeholder in the `index.html` file.
4.  Commit and push to the `main` or `master` branch. If GitHub Pages is enabled for this repository, it will automatically deploy your updates.
